# 01-NORTH_STAR.md: Intenção e Limites do Projeto

## 🎯 1. Intenção Central (The North Star)

R: Criar uma plataforma web para colecionadores de figurinhas da Copa do Mundo acompanharem o progresso de seus álbuns, encontrarem pessoas para troca e organizarem encontros de forma simples, rápida e segura.

## 😫 2. O Problema Real (A Dor)

R: Colecionadores frequentemente possuem muitas figurinhas repetidas e dificuldade para encontrar pessoas com quem trocar. Além disso, acompanhar manualmente o progresso do álbum é trabalhoso. O sistema centraliza essas informações e facilita a conclusão da coleção.

## 👥 3. Público-Alvo (As Personas)

O sistema é desenhado para atender às necessidades destes perfis:

Pedro (Colecionador Iniciante): Está montando seu primeiro álbum e quer visualizar facilmente quantas figurinhas faltam para completar a coleção.

Marina (Colecionadora Casual): Compra figurinhas ocasionalmente e deseja encontrar oportunidades de troca próximas de sua região.

Carlos (Colecionador Avançado): Possui muitas figurinhas repetidas e busca otimizar trocas para completar o álbum o mais rápido possível.

## 🔍 4. Checklist de Descoberta (5 Questões)

Fonte do Dado: Informações fornecidas pelos próprios usuários via navegador web.

Entrega: Atualização imediata do progresso do álbum e das oportunidades de troca.

Regra de Ouro: Facilitar a conexão entre colecionadores sem depender de grupos informais ou controle manual.

Resiliência: O sistema deve continuar funcional mesmo com infraestrutura simples.

Interface: Design moderno, responsivo e otimizado para celular e desktop.

## 🚫 5. Limites e Fora de Escopo

NÃO faremos integração com redes sociais.

NÃO criaremos aplicativos nativos para Android ou iOS.

NÃO implementaremos sistema de pagamentos.

NÃO utilizaremos frameworks pesados se HTML, CSS e JavaScript resolverem o problema.

## ⚖️ 6. Mindset de Simplicidade (Karpathy Style)

Este projeto segue a Regra 80/20: entregar o máximo valor com o mínimo de complexidade.

Foco: Cadastro de usuários, controle de figurinhas possuídas, figurinhas repetidas, busca de trocas e acompanhamento do progresso do álbum.

### 🛂 Instrução para a IA

Antes de iniciar qualquer implementação, garantir que as funcionalidades propostas respeitem os limites definidos neste documento e mantenham a simplicidade do MVP.
